The 0th term of the series is the constant term before the sum; $$\frac{a_0}{2}.$$ This is labelled as the 0th term so that the series may start indexing its terms from $n=1$. $$ f(x)=\frac{a_0}{2}+\sum_{n=1}^{\infty}a_n \cos\left(\frac{n\pi x}{L}\right)+b_n \sin\left(\frac{n \pi x}{L}\right) $$

If the function is not periodic, then we may cut the function, forcing it to be periodic over some $$ -L < x < L. $$ This means that we can prescribe a Fourier series that is now valid for the function over the selected range. If we take the limit as $L \rightarrow \infty$, then a Fourier transform need be used. This is a different problem entirely!
