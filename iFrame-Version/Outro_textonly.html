Fourier Transform

In a Fourier Series, if the range over which the function is periodic, $L$ goes to infinity, this function is no longer periodic. To treat such a function, we exercise Fourier Transform.
Visualisation Guide
On the right, we can see a square function, and a summation of sines and cosine functions with a finite number of terms that represents the square function. Try changing the number of terms in the Fourier Series. By increasing this, we can see that the function resembles closer and closer to the square function.
By the end of this suite, you would be able to understand the intuition behind Fourier Series, and to do it yourself for any reasonable periodic function.
